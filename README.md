### *Where speed meets data.*

**TheVelocityGrid** is a Formula 1-inspired web application that turns raw race data into an interactive experience.  
Built with **Python**, **FastF1**, **HTML**, **CSS**, and **JavaScript**, it brings you closer to the world of F1 through dynamic visuals, driver comparisons, and live stats.
